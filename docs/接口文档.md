# 接口文档

## 1. 认证说明

- 首次访问 `GET /` 返回密码页。
- 提交密码使用 `PHP Session` 建立会话。
- 登录页隐藏字段使用单次 Token。
- 连续 5 次登录失败将封禁来源 IP 10 分钟。

## 2. 通用错误码

| HTTP 状态码 | error | 含义 |
| --- | --- | --- |
| 400 | `BAD_REQUEST` | 请求参数格式错误 |
| 400 | `EMPTY_DIR` | 目录为空，无法打包 |
| 400 | `NOT_DIRECTORY` | 请求目标不是目录 |
| 401 | `UNAUTHORIZED` | 未登录或会话失效 |
| 403 | `INVALID_TOKEN` | Token 校验失败 |
| 404 | `NOT_FOUND` | 任务不存在 |
| 405 | `METHOD_NOT_ALLOWED` | 请求方法不支持 |
| 500 | `LIST_FAILED` | 目录读取失败 |
| 500 | `ZIP_FAILED` | 打包任务创建失败 |

## 3. 获取文件列表

### 请求

```http
GET /api/list HTTP/1.1
Cookie: CSFILESESSID=xxx
```

### 响应

```json
{
  "root": "1",
  "path": "",
  "type": "dir",
  "size": 0,
  "mtime": 1777000000,
  "items": [
    {
      "path": "软件包",
      "name": "软件包",
      "type": "dir",
      "size": 0,
      "mtime": 1777000100,
      "children": [],
      "empty": true
    },
    {
      "path": "demo.zip",
      "name": "demo.zip",
      "type": "file",
      "size": 123456,
      "mtime": 1777000200
    }
  ],
  "generated_at": 1777000300
}
```

### 字段说明

- `path`: 相对根目录路径
- `type`: `file` 或 `dir`
- `size`: 文件大小，目录固定为 `0`
- `mtime`: Unix 时间戳
- `children`: 子节点数组，仅目录存在
- `empty`: 目录是否为空

## 4. 单文件下载

### 请求

```http
GET /download.php?path=demo.zip HTTP/1.1
Range: bytes=0-1048575
Cookie: CSFILESESSID=xxx
```

### 响应头示例

```http
HTTP/1.1 206 Partial Content
Content-Type: application/octet-stream
Content-Disposition: attachment; filename="demo.zip"
Accept-Ranges: bytes
Content-Range: bytes 0-1048575/5368709120
Content-Length: 1048576
```

### 说明

- 支持 `HTTP Range Requests`
- 支持大文件断点续传
- 禁止下载 `.php`、`.py`、`.htaccess` 等敏感文件

## 5. 创建 ZIP 打包任务

### 请求

```http
POST /api/zip HTTP/1.1
Content-Type: application/json
Cookie: CSFILESESSID=xxx
```

```json
{
  "path": "子目录/素材",
  "csrf_token": "当前页面输出的会话令牌"
}
```

### 成功响应

首次创建任务：

```json
{
  "task_id": "4d5d8e0c4db9c03f6d0f5c7a8a6f...",
  "status": "running",
  "cached": false,
  "download_url": "/tmp/素材-9d12ab34cd56.zip",
  "signature": {
    "hash": "9d12ab34cd56...",
    "mtime": 1777000500,
    "count": 321
  }
}
```

缓存复用时：

```json
{
  "task_id": "4d5d8e0c4db9c03f6d0f5c7a8a6f...",
  "status": "done",
  "cached": true,
  "download_url": "/tmp/素材-9d12ab34cd56.zip",
  "signature": {
    "hash": "9d12ab34cd56...",
    "mtime": 1777000500,
    "count": 321
  }
}
```

### 失败响应

```json
{
  "error": "EMPTY_DIR",
  "message": "目录为空"
}
```

## 7. 进度流接口

### 请求

```http
GET /api/zip?stream=4d5d8e0c4db9... HTTP/1.1
Accept: text/event-stream
Cookie: CSFILESESSID=xxx
```

### SSE 推送示例

```text
event: progress
data: {"task_id":"4d5d8e0c4db9...","status":"running","percent":18,"speed":"10.24 MB/s","remaining":"30 s"}

event: progress
data: {"task_id":"4d5d8e0c4db9...","status":"running","percent":56,"speed":"11.88 MB/s","remaining":"12 s"}

event: progress
data: {"task_id":"4d5d8e0c4db9...","status":"done","percent":100,"speed":"12.01 MB/s","remaining":"0 s","download_url":"/tmp/demo.zip"}
```

### Python 脚本入参

PHP 启动 Python 进程时传入 JSON 字符串：

```json
{
  "source": "/www/wwwroot/cs/1/子目录",
  "target": "/www/wwwroot/cs/tmp/xxx.zip",
  "task_id": "任务ID",
  "compresslevel": 6,
  "memory_limit_mb": 512
}
```

### Python stdout 行格式

```json
{"percent": 0, "speed": "0 MB/s", "remaining": "--"}
{"percent": 35, "speed": "8.12 MB/s", "remaining": "24 s"}
{"percent": 100, "speed": "9.04 MB/s", "remaining": "0 s"}
```

## 8. 前端轮询策略

- 列表接口默认每 5 秒轮询一次
- 手动点击“立即同步”可强制刷新
- 若目录结构 JSON 未变化，则前端只更新时间戳，不重绘 DOM
